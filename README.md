# Pot Bot: protections of teams and groups

Bot z pełną rejestracją, CAPTCHA, anti-spam i bezpieczną bazą danych.

## Funkcje
- Rejestracja użytkowników (imię, nazwisko, numer telefonu);
- Działający system przeciw spamu;
- Weryfikacja CAPTCHA podczas rejestracji;
- Interakcja z bazą danych (SQLite).

## Uruchomienie
pip install -r requirements.txt
cp .env.example .env  # i wklej  API token
python main.py
